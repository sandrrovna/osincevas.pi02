Вариант 1. Напишите рекурсивную функцию для вычисления суммы цифр числа.
Описание алгоритма: идея заключается в том, чтобы найти сумму цифр числа. Для этого берём последнюю цифру (остаток от деления на 10) и прибавляем её к сумме цифр оставшейся части числа (целочисленное деление на 10). Базовый случай — когда число стало 0.

Пояснение работы алгоритма: 
n = abs(n) — преобразуем число в положительное с помощью функции abs(). Это позволяет корректно обрабатывать отрицательные числа: сумма цифр не зависит от знака.
if n < 10: return n — если число состоит из одной цифры, возвращаем её как результат. Это условие останавливает рекурсию.
last_digit = n % 10 — получаем последнюю цифру числа через остаток от деления на 10.
remaining_number = n // 10 — получаем число без последней цифры через целочисленное деление на 10.
return last_digit + sum_of_digits(remaining_number) — возвращаем сумму последней цифры и результата рекурсивного вызова для оставшейся части числа.

Временная сложность: O(n)

Почему: Каждый рекурсивный вызов уменьшает число примерно в 10 раз (деление на 10), поэтому количество вызовов пропорционально количеству цифр в числе.


Контрольный вопрос 10: Какая разница между обходом дерева в глубину (DFS) и в ширину (BFS)?
DFS (обход в глубину) идёт глубоко вниз по каждой ветви, прежде чем перейти к соседним вершинам.
BFS (обход в ширину) рассматривает узлы слой за слоем, начиная с самого верхнего уровня.
